# Varinda-Store-Sales-Analysis-Excel-Project

This project analyzes annual e-commerce sales data to identify key customer segments, purchasing patterns, and revenue drivers. The analysis is performed entirely in **Microsoft Excel**, using pivot tables, charts, slicers, and dashboards to explore trends and generate actionable business insights.

## Dataset used - [Download Excel File (Direct)](https://raw.githubusercontent.com/sluthra8833/Varinda-Store-Sales-Analysis-Excel-Project/main/Varinda%20Store%20Data%20Row%20File.xlsx)

## üìä Problem Statements & Key Questions

1. Compare **Sales vs Orders** over time.
2. Identify the **month with peak performance** in both sales and orders.
3. Determine **customer purchase behavior**: Men vs Women.
4. Analyze the distribution of **Order Status**.
5. Find the **Top 5 revenue-contributing states**.
6. Study how **Age and Gender influence order volume**.
7. Identify which **Sales Channel** contributes the most revenue.
8. Determine the **best-selling Product Category**.

## üîç Key Insights

- The business is **women-dominated** ‚Äì women contribute ~64% of all purchases.
- **March** shows the highest sales, after which sales gradually decline ‚Üí Possible seasonal demand or reduced marketing activity post-Q1.
- **Maharashtra, Karnataka, and Uttar Pradesh** contribute ~35% of total revenue.
- The **30‚Äì49 age group** forms the core customer base (~50% contribution).
- **Amazon, Flipkart, and Myntra** collectively account for ~80% of total sales.

## üéØ Data-Driven Recommendations to Improve Sales

1. **Expand Women-focused Product Range** (e.g., introduce new designs every 2 weeks).
2. Run **Instagram UGC/Reels campaigns** targeting women under 30.
3. Investigate the **March sales spike** and replicate similar promotional strategies.
4. Build a **Customer Retention List** (WhatsApp alerts, festival offers, loyalty drops).
5. Improve **product sizing details and descriptions** to **reduce return rates**.
